关于MetaScan智能体

🔬 系统简介

MetaScan是一款基于代谢组学技术的智能健康评估平台。用户通过录入代谢物浓度数据，系统自动比对参考范围，生成全面的代谢健康报告，帮助用户直观了解自身代谢状况。系统搭载DeepSeek AI智能助手，可提供专业的健康解读与个性化建议。

🧬 六大代谢通路覆盖

系统涵盖26种核心代谢物，分布于六大代谢通路：

🧬 氨基酸代谢
L-丝氨酸、L-谷氨酰胺、L-半胱氨酸、牛磺酸等
🍚 糖分代谢
D-葡萄糖、(R)-3-羟基丁酸、赤藓糖醇等
⚡ 能量代谢
苹果酸、α-酮戊二酸、丁二酸(TCA循环)
🛡️ 解毒与抗氧化
L-半胱氨酸、谷胱甘肽前体、含硫化合物
🫘 肾脏排泄功能
肌酐、肌酸、尿酸、嘧啶等含氮废物
💊 药物与毒素代谢
水杨尿酸、哌啶羧酸等外源物代谢

🎯 核心功能

📝
代谢物浓度直接录入
26种代谢物全部展示，用户自由填写浓度值。实时比对参考范围，偏高偏低一目了然
📊
多维度可视化报告
雷达图、柱状图、环形图、偏离度瀑布图——8种图表直观展示代谢健康状况
🤖
DeepSeek AI智能助手
接入大语言模型，提供代谢数据专业解读、健康建议与在线问答
📅
历史数据追踪
自动保存每次检测记录，支持按时间筛选、导出CSV，长期追踪代谢变化趋势
💪
个性化健康方案
根据代谢数据自动生成饮食建议、运动方案及干预时间线
📥
报告导出与分享
支持导出PDF、Word格式，生成健康卡片，一键分享检测结果

🖥️ 技术架构

Vue 3 + Vite
前端框架与构建工具
Chart.js
数据可视化图表引擎
DeepSeek API
大语言模型智能分析
Express + WebSocket
后端服务与实时通信
localStorage
本地数据持久化存储
Tesseract OCR
化验单拍照识别引擎
